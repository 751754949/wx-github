## 本地开发、

> 默认你会用`vue`

1. 先大概过一遍[微信小程序的文档](https://developers.weixin.qq.com/miniprogram/dev/)
2. 再认真仔细看一遍[mpvue的文档](http://mpvue.com/mpvue/#_1)
3. 有兴趣的话看看[github REST API v3文档](https://developer.github.com/v3/)

```javascript
npm i
npm run dev

npm run build // 打包
```
